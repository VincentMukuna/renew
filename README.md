# Reneww

Reneww is a private, local-first subscription tracker in the Renew personal finance tools suite.

The visible product name is **Reneww**. Technical identifiers use `renew`.

## Development

```sh
npm install
npm start
```

Start with Expo Go for this managed foundation. Native folders are intentionally not committed yet; run `npx expo run:ios` or `npx expo run:android` only when a feature needs a custom native build.
